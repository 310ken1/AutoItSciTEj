ソフト名称：
	hhh.exe


取り扱い種別：
	フリーソフト


ソフトの概要：
	キーワードに対応した、HTMLヘルプ表示プログラムです。

	このプログラムを使うと、TeraPad や EmEditor など外部ＨＴＭＬヘルプに対応していないエディタでも、
	外部ＨＴＭＬヘルプに対応させる事が出来ます。
	
	マイクロソフトのHTMLヘルプを表示するプログラムは、
	hh.exe（Microsoft HTML Help Executable）ですが、このプログラムは、キーワードに対応していません。
	そこで、キーワードに対応した hh.exe と言う事で、プログラム名、hhh.exe を作成しました。
	                                                                ~

インストール手順：
	プログラムを解凍し、適当な場所にコピーします。
	プログラムをコピーした場所が環境変数PATHが通らない位置の場合、
	プログラム名に絶対パス(C:\・・・\hhh.exe、ドライブ･レター以降〜プログラム名まで)を指定する様にします。

	使い方： hhh.exe　"〜.chm"　Keyword  で、
	         外部HTMLヘルプを開き、キーワードの表示を行います。
